# First Virtual Company

Mission :

Concevoir et développer un récepteur ADS-B basé sur un SDR ADALM-Pluto.

Objectifs :

- Recevoir les trames ADS-B à 1090 MHz
- Décoder les messages Mode-S
- Afficher les aéronefs détectés
- Étudier les performances radio
- Construire une architecture réutilisable

Horizon :

Prototype fonctionnel en 3 mois.
